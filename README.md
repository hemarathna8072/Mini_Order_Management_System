Mini Order Management System 

📌 Project Overview
This project is a Mini Order Management System built using the MERN stack. It allows users to browse products, place orders, and manage order status.

⚙️ Backend Setup
Navigate to backend folder - cd backend
Install dependencies - npm install
Configure Environment Variables - Create a .env file in the backend folder and add: PORT=5000 MONGO_URI=your_mongodb_connection_string JWT_SECRET=your_secret_key
Run the backend server - npm start

💻 Frontend Setup
Navigate to frontend folder - cd frontend
Install dependencies - npm install
Run the frontend - npm start
